# Professional Sports Projection System

An AI-driven proprietary projection model designed to calculate fair odds and spot Expected Value (+EV) edges in sports betting markets.

## Deployment on Streamlit Cloud
1. Push this entire folder repository to GitHub.
2. Log into [Streamlit Share](https://share.streamlit.io/).
3. Connect your repository and select `streamlit_app.py` as your Main file path.
4. Deploy! The app will automatically fetch data files on initial startup using `kagglehub`.
